# LectureEvaluation
강의 평가를 위한 사이트 개발

## 개발 언어 및 환경
jsp, java, mysql, bootstrap, eclipse

## Model

### likey
- 좋아요 기능 모델링
### evaluation
- 평가 모델링
- 글의 삭제 및 등록 처리
### user
- 사용자 정보 모델링
- 로그인, 회원가입, 이메일 관련 처리
### util
- DB 연동
- Gamil id, pw 값 처리
- sha256을 이용한 해싱처리
